# Outreach Packs — Top 10 (sales-ready, first-touch)

Companion to `outreach_tracker.tsv` (live status) and `bali_villa_prospects.csv` (full ranking).
Send order below is the **execution order** (section: Recommended sending order). The Rank column
in the priority table is a value ranking, not the send order — send in the numbered order.

Guardrails carried from `outreach_scripts.md`: calm, specific, evidence-led, no hype, no fake
urgency, no mass-blasting. Every message references something real and public. A single test is a
signal, not proof of systemic failure. Max 4 touches per lead, then park 60 days.

## Two CTA variants
- **Cold-open (soft) CTA** — for the very first cold message: offer the audit you already did, then
  a soft next step ("I can run the deeper Secret Guest Audit through your public inquiry flow and
  send the report by WhatsApp"). Lower friction; recommended for the First 3.
- **Direct CTA** — "send me one anonymized real guest inquiry and I'll map the exact flow." Slightly
  bigger ask; used in the full packs below.

Use judgment per lead. Both are captured so nothing is lost.
